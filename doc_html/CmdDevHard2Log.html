Command DevHard2Log :
    Returns the logical device/logical channel of a memory location
    on wago controller



DevHard2Log Definition
Input Argument Tango::DEVVAR_SHORTARRAY [0] : MSB=I/O LSB=Bit/Word (ex: 0x4957 = (`I`<<8)+`W`)
[1] : offset in wago controller memory (ex: 0x16)
Output Argument Tango::DEVVAR_SHORTARRAY [0] : logical device key
[1] : logical channel
